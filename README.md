# **Airline Dashboard**

A simple Flask-based web application to visualize airline performance data and flight statistics.

---

## **Tech Stack**

Backend: Python, Flask
Frontend: HTML, CSS, JavaScript
Charts: Chart.js / Plotly

---

## **Key Features**

* View flight performance statistics
* Delay & cancellation analysis
* Passenger and revenue insights
* Interactive charts and data visualization
* Search & filter options

---

## **Getting Started**

**Clone the repo:**

```
git clone https://github.com/Sujimirutikaa/airline-dashboard.git
```

**Install requirements:**

```
pip install -r requirements.txt
```

**Run the app:**

```
python app.py
```

---

## **Project Structure**

**templates/** – HTML pages
**static/** – CSS, JS, images
**data/** – Airline data files (CSV)
**app.py** – Main Flask application
